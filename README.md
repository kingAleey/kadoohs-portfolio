# King Kadooh — Portfolio

A personal portfolio site for King Kadooh — Web Developer, Cyber Security Specialist, Bybit affiliate, and Web3/DeFi enthusiast. Showcases platforms/socials, work history, resume, skills, and a contact form.

Built with plain HTML, CSS, and JavaScript — no build step, no framework, no dependencies. Deployed on Netlify.

## Pages

- `index.html` — home page with hero, social platforms, about, skills, and interests
- `work.html` — a grid of past projects and work
- `resume.html` — career summary, work experience, education, and skills
- `contact.html` — a contact form powered by Netlify Forms

## Running locally

No build tools are required. Open `index.html` directly in a browser, or serve the folder with any static file server, e.g.:

```
npx serve .
```
